What you can do?

I'm soon going to start working on TitanOS (or Linux From Scratch) and hopefully make it into a campus-wide project that can last for generations and hopefully revitalize the IMSA Computer Science program.

Why?

Imagine a place where collaboration is truly at the center of attention. Where innovation and experimentation are encouraged. Although it will have experimentation, it will likely follow similar development as the actual Linux kernel so that there are some usable versions. After some time, this can hopefully be expanded so that we can teach Android ROM Development and hopefully get some sort of Android Program running here at IMSA.

Future:

  1. Create a usable OS maintained by IMSA
  2. Make custom ROMs/Applications for Android
  3. Put more emphasis on Linux in general at IMSA
  4. Show the world that IMSA is willing and capable
